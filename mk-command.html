Plain English First

Terms You'll See Here

Orchestration ▼

Coordinating multiple AI agents to work together — like a conductor keeping all musicians in sync so they produce one great performance instead of noise.

System Prompt ▼

The job description you give your AI before it starts. It tells the AI who it is, what it does, and how it should behave for every response.

Workflow ▼

A set of steps that happen in a specific order, automatically — like a recipe your AI follows every single time without you having to re-explain it.

LLM ▼

Large Language Model — the brain inside Claude and ChatGPT. It was trained on enormous amounts of text and can write, reason, and answer questions like a human.

Token ▼

How AI measures text — roughly 1 token = 1 word. A 100-word Instagram caption is about 130 tokens. Free Claude accounts have daily token limits.

API ▼

A connection between two apps — like a phone call between software. When Buffer auto-posts to Instagram, it's using Instagram's API to deliver the content.

The Daily Briefing
🛡️ GUARDIAN

Start every day by asking GUARDIAN to audit yesterday's results. This single habit compounds into dramatically better content over time — you'll know exactly what to repeat and what to kill.

⚡ Command PromptiThe instructions you give your AI before it starts working — like a job description 
GUARDIAN, run my morning audit. Review [yesterday's metrics — paste your numbers here]. Tell me: top 3 performing pieces and why they worked, what to repeat this week, what to stop doing immediately, and today's content focus theme.
2
The Content Machine
🎨 NOVA

One command. Full content suite for all your platforms. NOVA knows your voice, your audience, and your niche — just give it the day's theme and let it produce.

⚡ Command Prompt 
NOVA, create today's content suite: 1 main post for each platform (Instagram, TikTok, Twitter/X, LinkedIn). Theme: [TODAY'S THEME]. Hook first on every piece. Include platform-specific formatting and a CTA on each. Write in my brand voice.
3
The Video Brief
🎬 PULSE

Get a complete, ready-to-shoot video package — script, shot list, b-roll suggestions, and teleprompter formatting. You just show up and press record.

⚡ Command Prompt 
PULSE, script my video for today. Topic: [TOPIC]. Platform: [PLATFORM]. Length: [LENGTH — 15s / 30s / 60s]. Include: shot-by-shot script with [SHOT] [ACTION] [DIALOGUE] format, teleprompter script, B-roll shot list, and hook options (give me 3 hooks to choose from).
4
The Growth Command
📲 ARIA

Hand your approved content to ARIA and it schedules, hashtags, and queues everything. Also gets 3 community engagement replies written so you look active without spending hours in the comments.

⚡ Command Prompt 
ARIA, schedule this week's content. Here's my content: [PASTE YOUR APPROVED CONTENT]. Assign optimal posting times per platform. Add 30-hashtag sets for each post. Write 3 community engagement reply templates I can use in my comments today.
5
The Money Command
🎯 REACH

When it's time to run paid ads or launch an offer, REACH writes 3 complete ad variations in direct-response format. Five hooks, tested frameworks, built-in urgency — ready to plug into Ads Manager.

⚡ Command Prompt 
REACH, write 3 ad variations for [PRODUCT/SERVICE]. Target audience: [AUDIENCE]. Budget level: [BUDGET]. Format: [Facebook Feed / TikTok Spark / Instagram Story]. Include: 5 hook options, primary text for each variation, headline, description, and a strong CTA. Use AIDA framework on variation 1, PAS on variation 2, Before-After-Bridge on variation 3.
