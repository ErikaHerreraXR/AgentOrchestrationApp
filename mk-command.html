What Does This Mean?

Terms you'll see — click to expand

Orchestration ▼

Coordinating multiple AI agents to work as a team — like a conductor keeping all musicians in sync so they produce one result.

System Prompt ▼

The job description you give your AI before it starts — tells it who it is, what it does, and how to behave for every response.

Workflow ▼

Steps that happen in a specific order, automatically — like a recipe your AI follows every time without needing re-explanation.

LLM ▼

Large Language Model — the brain inside Claude and ChatGPT. It was trained on billions of text examples and can write, reason, and create.

Token ▼

How AI measures text — ~1 token = 1 word. A 100-word caption is about 130 tokens. Free AI plans have daily token limits.

API ▼

A connection between two apps — like a phone call between software. Buffer uses Instagram's API to post for you automatically.

Run Everything With 5 Prompts

These are the only commands you need. Check each one as you run it. Every check builds your streak. 🔥

01
The Morning Audit
🛡️ GUARDIAN
⏱ 10 min +20 XP

Start every day knowing exactly what worked yesterday, what to repeat, and what to kill. GUARDIAN gives you a data-driven brief in minutes — not hours.

⚡ Command Prompt 
GUARDIAN, run my morning audit. Review [yesterday's metrics — paste your numbers here]. Tell me: top 3 performing pieces and why they worked, what to repeat this week, what to stop doing immediately, and today's recommended content focus theme.
02
The Content Machine
🎨 NOVA
⏱ 15 min +20 XP

One command. Full content suite for all platforms. NOVA knows your voice — just give it the day's theme and let it produce at scale.

⚡ Command Prompt 
NOVA, create today's content suite: 1 main post for each platform (Instagram, TikTok, Twitter/X, LinkedIn). Theme: [TODAY'S THEME]. Hook first on every piece. Include platform-specific formatting and a CTA on each. Write in my brand voice.
03
The Video Brief
🎬 PULSE
⏱ 5 min +20 XP

A complete ready-to-shoot video package — script, shot list, teleprompter, B-roll. You just show up, press record, and follow directions.

⚡ Command Prompt 
PULSE, script my video for today. Topic: [TOPIC]. Platform: [PLATFORM]. Length: [15s / 30s / 60s]. Include: shot-by-shot script in [SHOT] [ACTION] [DIALOGUE] format, teleprompter script, B-roll shot list, and 3 hook options to choose from.
04
The Growth Schedule
📲 ARIA
⏱ 10 min +20 XP

Hand approved content to ARIA and it schedules, hashtags, and queues everything. Also preps 3 community replies so you look active without hours in comments.

⚡ Command Prompt 
ARIA, schedule this week's content. Here's my content: [PASTE YOUR APPROVED CONTENT]. Assign optimal posting times per platform. Add 30-hashtag sets for each post. Write 3 community engagement reply templates I can use in my comments today.
05
The Money Command
🎯 REACH
⏱ 5 min +20 XP

When it's time to run paid ads or launch an offer, REACH writes 3 complete variations in direct-response format — 5 hooks, proven frameworks, built-in urgency.

⚡ Command Prompt 
REACH, write 3 ad variations for [PRODUCT/SERVICE]. Target: [AUDIENCE]. Budget level: [BUDGET]. Format: [Facebook Feed / TikTok Spark / Instagram Story]. Include: 5 hook options, primary text for each variation, headline, description, strong CTA. Use AIDA on #1, PAS on #2, Before-After-Bridge on #3.
