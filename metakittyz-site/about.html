04 — the real build

The Full Truth: What Broke & How We Fixed It

This is not a highlight reel. Every section of the Metakittyz build hit real problems. We're documenting all of them here — not to look impressive, but because watching something break and get fixed is the actual education. If you're building your own AI system, these are the exact errors you'll hit too.
01
Website — Video Embeds Were Completely Broken
✅ RESOLVED — watch.html + index.html fixed
▾
⚠️ The Problem
  • ❌YouTube embed used live_stream?channel=UCmetakittyz — not a valid embed URL
  • ❌Twitch both player and chat iframes had literal text DOMAIN_PLACEHOLDER that was never replaced
  • ❌YouTube live chat used channel URL instead of specific video ID
✅ The Fix
  • →YouTube embed: changed to youtube.com/embed/{VIDEO_ID}?autoplay=1
  • →YouTube chat: changed to live_chat?v={VIDEO_ID}&embed_domain=metakittyz.com
  • →Twitch: replaced DOMAIN_PLACEHOLDER with parent=metakittyz.com in both iframes
💡 Lessons Learned
  • →YouTube requires a specific video ID, not a channel ID, for embeds to work
  • →Twitch's parent= must exactly match the live domain — no wildcards, no placeholders
  • →Update the YouTube video ID before each stream — build this into the pre-stream checklist
02
Agent System — 10 Agents Was Too Many
✅ RESOLVED — trimmed to 6 core agents
▾
⚠️ The Problem
  • ❌Started with 10 agents: MATRIX, NOVA, SPARK, PULSE, SIGNAL, SCHOLAR, ECHO, REACH, ARIA, GUARDIAN
  • ❌Too complex to explain on stream — confused new viewers
  • ❌SCHOLAR was specific to Harvard course, SIGNAL and ECHO overlapped with PULSE and ARIA
  • ❌Inconsistency between playbook (showing 10) and site (should show 6)
✅ The Fix
  • →Trimmed to 6 essential agents: MATRIX, NOVA, SPARK, PULSE, ARIA, GUARDIAN
  • →Each agent now has a clearly distinct role — no overlap
  • →Updated Kitty Lab dropdown, system prompts, templates, and playbook to match exactly
  • →Pipeline flow diagram added: MATRIX → NOVA → SPARK → PULSE → ARIA → GUARDIAN
💡 Lessons Learned
  • →Start with the minimum viable agent set. Add agents when a gap is proven, not assumed
  • →Every agent needs a unique, non-overlapping role or it creates confusion
  • →Consistency across every file matters — one number wrong = trust broken
03
Kitty Lab — API Has No Live Fallback Yet
⚙️ IN PROGRESS — Netlify Functions being built
▾
⚠️ The Problem
  • ❌Kitty Lab calls /api/anthropic and /api/openai — but these Netlify Functions don't exist yet
  • ❌Right now, running a prompt gives an API error unless the user copies the prompt into Claude/ChatGPT manually
  • ❌The fallback shows a graceful error message, but the full live pipeline isn't wired yet
✅ The Fix (In Progress)
  • →Building netlify/functions/anthropic.js with Claude claude-opus-4-8
  • →Building netlify/functions/openai.js as fallback with GPT-4o
  • →+3 retry with 2s/4s/8s exponential backoff implemented in the cascade logic
  • →Deploying live on stream — watch it get built in real time
💡 Lessons Learned
  • →Ship the front-end first so users can use it immediately (copy prompt to Claude)
  • →A graceful error message that tells users what to do is better than a silent failure
  • →Never ship something that silently fails — always surface the error with context
04
Streaming — Getting Twitch + YouTube Running Simultaneously
✅ RESOLVED — Restream.io configured
▾
⚠️ The Problem
  • ❌OBS can only stream to one RTMP endpoint natively
  • ❌Twitch and YouTube require separate RTMP ingest URLs
  • ❌Splitting manually means double the encoder load and sync issues
✅ The Fix
  • →Restream.io Professional ($41/mo) handles the split — one OBS output, two platforms
  • →OBS → Restream RTMP → splits to Twitch + YouTube simultaneously
  • →Both platforms get the same stream at the same quality
💡 Lessons Learned
  • →Don't try to DIY multi-platform streaming — the tools exist and aren't expensive
  • →The $41/mo is worth it vs. the engineering time to build a custom solution
  • →Test the stream at least 10 minutes before going live — not 2 minutes before
05
Site Navigation — Pages Were Invisible to Each Other
✅ RESOLVED — sticky nav added to all pages
▾
⚠️ The Problem
  • ❌guide.html, community.html, and kitty-lab.html existed but had no links to each other or to index.html
  • ❌Users had to know the exact URL to find each page — there was zero discoverability
  • ❌index.html had no site navigation at all — just the hero page
✅ The Fix
  • →Added sticky `.site-nav` to index.html with tabs: Home / Guide / Community / Kitty Lab / About
  • →Added cross-page links to guide.html and community.html matching their color schemes
  • →Built kitty-lab.html with the same nav system pre-wired from the start
💡 Lessons Learned
  • →Navigation is not a feature — it's the foundation. Build it first, not last
  • →A page that can't be found from the site might as well not exist
  • →Every page should link to every other page — no dead ends
