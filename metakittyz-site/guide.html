Quick Reference
Start here before Episode 1. Everything on this page is free to install. No coding experience needed — just follow each step in order and check them off as you go.
STEP 0 — BEFORE YOU WATCH
Set up your dev environment
Your computer needs 5 free tools before you can build your first agent. Install them in the order shown. The whole process takes about 20–30 minutes.
1. Node.js
The engine that runs your agent code. Download the LTS (long-term support) version — it is the most stable. After installing, open your Terminal and type node --version to confirm it worked.
nodejs.org →
2. VS Code
A free code editor — like Word, but for writing agent instructions. Used by millions worldwide. After installing, open it and you will see a welcome screen.
code.visualstudio.com →
3. Git
Saves different versions of your project so you can undo mistakes at any point. Think of it as unlimited undo for your entire project. After installing, type git --version in your Terminal.
git-scm.com →
4. GitHub account
Where your project files live online so you can access them from anywhere. Free for personal projects. Sign up with your email and choose a username.
github.com →
5. Anthropic account
You need an API key — a private password that lets your code talk to Claude. Free trial credits are included when you sign up. Go to the Console tab after signing in.
console.anthropic.com →
Claude or ChatGPT
You will use AI to help write and debug your code throughout the series. A free account on either is enough. Keep a tab open while watching.
claude.ai →
KEY TERMS — KNOW THESE BEFORE EPISODE 1
APIApplication Programming Interface. A way for two programs to talk to each other. When your agent sends a message to Claude, it uses an API. Think of it as a drive-through window — you place an order, the kitchen prepares it, and you get a response back.
API keyA long string of letters and numbers that proves you are allowed to use a service. Treat it like a password — never share it, never paste it into a public file, and never post it online.
TerminalA black box where you type commands to your computer instead of clicking buttons. On Mac it is called Terminal. On Windows it is Command Prompt or PowerShell. You will use this to install things and run your agent.
npmNode Package Manager. A tool that comes with Node.js and lets you install code libraries with one command. Typing "npm install" in your Terminal downloads everything your project needs automatically.
.env fileA hidden file in your project where you store secret information like your API key. Your code reads from this file instead of having the key written directly in your code — which keeps it safe if you share your code.
TypeScriptThe coding language used in this series. It is JavaScript (the language of the web) with extra safety features that catch mistakes before they cause problems. No prior experience needed — Erika teaches it from scratch.
AgentAn AI that can take actions, not just answer questions. It can search the web, write posts, check results, and decide what to do next — all automatically. The goal of this series is to build a team of 6 of them.
PromptThe instructions you give to an AI. A well-written prompt is the difference between a generic response and a perfect one. The entire series is built around mastering this skill — without any code required.
TokenThe unit of text an AI reads and writes. Roughly 1 token = 0.75 words. APIs charge by token, so understanding tokens helps you estimate and control your monthly costs.
MCPModel Context Protocol. A standard way to connect your agent to external tools like calendars, databases, or social media. Think of it as a universal plug adapter that lets your AI work with any app. Covered in Part 3.
