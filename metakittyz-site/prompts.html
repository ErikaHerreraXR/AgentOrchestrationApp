Step 00
Production Workflow
The exact sequence from zero to published marketing content. Follow this order every time for production-ready results.
1
Define Product + Offer
Write your product name, price, key benefit (one sentence), and target customer before touching any AI tool. This becomes the brief every prompt is built on.
2
Generate Base Avatar
Create your AI spokesperson using the Avatar Base prompts. Save the seed/reference image — this is your character's identity card used in every subsequent generation.
3
Generate All Angle Variants
Produce 8–12 angle variants (headshots + body shots) using consistency prompts. All must match the base character before moving forward.
4
Produce Marketing Ad Video
Use Higgsfield or budget tool prompts to animate your avatar into a product ad. Hook → Problem → Solution → CTA structure. 15–30 seconds.
5
Add Voice + Audio
Generate voiceover using ElevenLabs or Higgsfield voice. Layer background music (royalty-free). Mix to -14 LUFS for social platforms.
6
Format Per Platform
Export in 9:16 (Reels/TikTok/Shorts), 1:1 (Instagram feed), 16:9 (YouTube). Add captions, on-screen text, and CTAs per platform spec.
7
Distribute + Sell
Schedule across all platforms. List the prompt pack as a digital product on metakittyz.com. Track CTR, saves, and conversions per piece.

Step 05 · Avatar Creation
Character Consistency System
The biggest challenge in AI avatar marketing is keeping the same character across 50+ images and videos. Follow this protocol exactly.
The Consistency Rule: Every generation that uses your avatar must include the exact same core descriptors — copy-paste them at the start of every new prompt. Never rephrase them. Midjourney's --cref flag + same descriptors = 85–95% visual consistency.
Character Anchor Block — Copy-Paste Into Every Prompt
Consistency Key
Your Character Anchor (fill in once, use everywhere)
ANCHOR TEMPLATE — replace brackets with your character's specifics: [AGE]-year-old [GENDER], [SKIN TONE] skin, [HAIR DESCRIPTION], [EYE COLOR] eyes, [DISTINGUISHING FEATURE e.g. small nose ring / sharp cheekbones / strong brow], [BODY TYPE: slim / athletic / curvy / stocky], always wearing [SIGNATURE COLOR ELEMENT e.g. hot pink accessories / gold jewelry / dark minimalist clothing] EXAMPLE FILLED ANCHOR: 28-year-old woman, warm brown skin, long dark wavy hair with highlights, dark brown almond eyes, strong defined cheekbones, slim athletic build, always wearing hot pink or gold accents, natural makeup, clear glowing skin
Consistency Protocol
Save anchor block in a notes doc you open before every session
Use --cref with your best base image every time in Midjourney
Keep --cw between 70–85 (lower = more creative, higher = more locked)
Regenerate any image where face differs by more than 10% before moving forward
Build a reference sheet: 6 approved angles saved in one folder labeled "CANON"
