STEP-BY-STEP DEV SETUP

Complete these steps before Episode 1. Every tool is free. If you get stuck on any step, post in the Discord #tech-help channel and someone will help you.

1
Install Node.js
Node.js is the engine that runs your agent code on your computer. Go to nodejs.org and download the version labelled LTS (long-term support). Run the installer and click through the default options.
node --version
Type this in your Terminal after installing. You should see a version number like v20.x.x — that means it worked.

nodejs.org →
2
Install VS Code
VS Code is a free code editor — like Microsoft Word, but for writing agent instructions and code. Download it from the link below, install it, and open it. You will see a welcome screen with a few options — you can ignore them for now.
code.visualstudio.com →
3
Install Git
Git saves different versions of your project so you can undo mistakes at any point. Think of it as unlimited undo for your entire project — not just the last change. Download from the link below and run the installer with default settings.
git --version
Type this in your Terminal after installing. You should see "git version 2.x.x".

git-scm.com/downloads →
4
Create a GitHub account
GitHub is where your project files live online — like Google Drive, but for code. Free for personal projects. Sign up with your email and choose a username. You will connect this to your project in Episode 1.
github.com →
5
Get your Anthropic API key
This is the private key that lets your code talk to Claude. Go to console.anthropic.com, sign up for a free account, and navigate to the API Keys section. Click "Create Key" and copy the key somewhere safe — you will only see it once.
ANTHROPIC_API_KEY=sk-ant-...
You will paste your real key here inside a file called .env in your project. Never share this key or post it online.

console.anthropic.com →
6
Create your first project folder
Open VS Code. Click "Open Folder" and create a new empty folder called metakittyz-agent somewhere on your computer. Open that folder in VS Code. Then open the Terminal inside VS Code by pressing Ctrl+` (backtick) on Windows or Cmd+` on Mac.
npm init -y && npm install @anthropic-ai/sdk dotenv
Paste this into your VS Code Terminal. It sets up your project and installs the Anthropic SDK in one command. You will see files appear in your folder.
